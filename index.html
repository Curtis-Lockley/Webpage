HTML And CSS




There are three methods to implementing CSS into your webpage. The three methods are inline, internal and external. As you can seefrom the poster on the left, each type of implementation has their pros and cons and it is important to know when to use each method. Inline is when you want to style an individual elemenet. Internal is where all the CSS code is implemented in the same document as the HTML code. Lastly, external is where the HTML document will link itself to an external document that will contain all of the CSS code. If you're wondered as to which method you should use, I present the pros and cons of each implementation method further down this page.
